Service Descriptions

Docker is a containerization platform that allows you to run applications in isolated environments called containers. This allows you to run multiple applications on the same machine without having to worry about them interfering with each other.

For this project, I've set up a Docker Compose file that will run the following containers:

    NGINX

    NGINX is a web server that can be used as a reverse proxy. This means that it can forward requests to other servers based on the request's URL. In this case, NGINX is forwarding requests to the WordPress container.

    WordPress + php-fpm

    WordPress is a content management system that allows you to create and manage websites. It uses php-fpm to process PHP code.

    MariaDB

    MariaDB is a relational database management system that is used by WordPress to store data.

    Redis

    Redis is an in-memory data structure store that is used by WordPress to cache data.

    FTP server

    The FTP server is used to upload files to the WordPress container.

    Adminer

    Adminer is a web-based database management tool that is used to manage the MariaDB database.

    Cadvisor

    Cadvisor is a container monitoring tool that is used to monitor the Docker containers.

    This webpage

    This webpage is used to provide an overview of the infrastructure.

These containers are all linked together using a Docker network. This allows them to communicate with each other.
